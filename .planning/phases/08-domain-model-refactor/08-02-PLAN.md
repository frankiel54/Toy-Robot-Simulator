Rewrite the three test files whose existing assertions targeted the API removed in plan 08-01.

Purpose: Phase 8 removed three Simulator pass-through properties (X, Y, Facing), four Robot mutation methods (PlaceAt, MoveTo, TurnLeft, TurnRight), and the Direction.Unset sentinel. - ToyRobot.Tests/SimulatorTests.cs — all X/Y/Facing assertions replaced with Report() string assertions
